外语学习辅助工具，在观看双语字幕时，可以用黑块儿遮挡字幕来观看

## 📚 使用方法

（单词/多次）点击浏览器上的插件图标，即可（启动/关闭）插件

## 🌟 功能特性

- 🖱️ 支持手动拖拽拉伸黑块儿的位置、大小
- 💾 黑块儿的位置大小信息，会根据网页域名来缓存到浏览器里
- 🔤 按住键盘 `s` 可以隐藏

## ⚠️ 注意事项

为了让黑块儿的交互体验更好，启动功能时，整个页面会覆盖一个透明的遮罩层。以至于页面的所有内容处于不可点击状态；可以使用键盘快捷键，来配合观看
